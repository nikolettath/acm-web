# acm-web

Simple Vite project.

## setup

1. install Node.js (LTS):
   https://nodejs.org/

check it worked:

```bash id="t1y2kp"
node -v
```

2. download the project from GitHub (ZIP) and extract it

3. open a terminal inside the project folder and run:

```bash id="w6s8nq"
npm install
```

4. start the project:

```bash id="a4m9cz"
npm run dev
```

open the link shown in the terminal (usually http://localhost:5173)

## build

```bash id="g7k2jd"
npm run build
npm run preview
```
